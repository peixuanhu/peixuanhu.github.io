# 排序不等式

## 模板题

### 排队打水

#### 思路

总等待时间：`t1 * (n-1) + t2 * (n-2) + ... + tn * 0`

时间长的人放在后面，时间短的人放在前面

#### 代码

```cpp
#include<iostream>
#include<algorithm>

using namespace std;

typedef long long LL;

const int N = 100010;
int n, t[N];

int main()
{
    scanf("%d", &n);
    
    for (int i = 0; i < n; i ++) scanf("%d", &t[i]);
    
    sort(t, t + n);
    
    LL res = 0;
    for (int i = 0; i < n; i ++) res += t[i] * (n - i - 1);
    
    printf("%lld", res);
    
    return 0;
}
```
