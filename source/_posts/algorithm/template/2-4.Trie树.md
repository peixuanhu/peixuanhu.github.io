# Trie树

## 适用场景

是用于高效**存储**和**查找**字符串集合中各字符串出现次数的数据结构。

## 代码模板

```cpp
int son[N][26], cnt[N], idx;
// 0号点既是根节点，又是空节点（一个点如果没有子节点，它的son就指向0）
// son[][]存储树中每个节点的子节点
// cnt[]存储以每个节点结尾的单词数量

// 插入一个字符串
void insert(char str[]) {
  int p = 0;
  for (int i = 0; str[i]; i ++) {
    int u = str[i] - 'a';
    if (!son[p][u]) son[p][u] = ++ idx; // 如果没有子节点，创建一个新节点继续走
    p = son[p][u];
  }
  cnt[p] ++;
}

// 查询字符串出现的次数
int query(char str[]) {
  int p = 0;
  for (int i = 0; str[i]; i ++) {
    int u = str[i] - 'a';
    if (!son[p][u]) return 0;
    p = son[p][u];
  }
  return cnt[p];
}
```
